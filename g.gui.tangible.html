DESCRIPTION

Tangible Landscape is a collaborative modeling environment for analysis of terrain changes which couples a scanner, projector and a physical 3D model with GRASS GIS.

The software consists of a plugin for GRASS GIS and addon r.in.kinect for Kinect for Windows.
